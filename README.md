## 🚀 Quick Start

```bash
# Clone repository
git clone <repository-url>
cd zahara-v1

# Start all services
cd infra && make up

# Run tests
make test-python

# Access the application
# Frontend: http://localhost:3001
# API: http://localhost:8000
# Router: http://localhost:7000
```
